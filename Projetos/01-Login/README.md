## Projeto 01 - Tela de Login

Este projeto consiste em uma tela de login estática, desenvolvida exclusivamente com HTML5 e CSS3, com foco em estruturação semântica e estilização responsiva.

A interface contempla apenas dois campos essenciais para autenticação:

- Campo de e-mail
- Campo de senha
- Botão de entrar

O objetivo do projeto é demonstrar a aplicação de boas práticas de marcação HTML e organização de estilos em CSS, priorizando clareza visual, usabilidade e design limpo.

Não há integração com backend ou validação dinâmica, tratando-se de uma implementação front-end básica para fins educacionais e portfólio.
